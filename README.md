# Mini Enterprise Project 🚀

## 🔧 Backend (FastAPI)

* JWT Authentication
* Role-Based Access (Admin, Manager, User)
* Task Management System

## 🎨 Frontend (React)

* Login Page
* Register Page
* Dashboard

## ⚙️ Tech Stack

* FastAPI
* SQLAlchemy
* MySQL
* React.js
* Tailwind CSS

## ▶️ Run Backend

cd backend
uvicorn app.main:app --reload

## ▶️ Run Frontend

cd frontend
npm install
npm run dev

## 📌 Features

* Secure Login using JWT
* Role-based API access
* Task assignment system
